# 💬 Real-time Chat

Zamonaviy real-time chat ilovasi. Hech qanday backend kerak emas!

## ✨ Xususiyatlar

- 🎨 Zamonaviy dizayn (gradient, animatsiyalar)
- 👤 Foydalanuvchi tanlovi (avatar emoji bilan)
- 💬 Xabarlar tarixi (localStorage)
- 🤖 Bot suhbatdosh (demo)
- ⌨️ "Yozmoqda..." ko'rsatkichi
- 🔄 Boshqa tablar bilan sinxron (storage event)
- 📱 To'liq responsive
- 💾 Xabarlar saqlanadi

## 🚀 Ishlatish

1. **Ism kiriting** — o'zingizga avatar tanlang
2. **Chat'ga kirish** — bosing
3. **Xabar yozing** — bot javob beradi
4. **2 ta tab oching** — xabarlar sinxron ko'rinadi!

## 🛠 Texnologiyalar

- HTML5
- CSS3 (animations, gradients)
- Vanilla JavaScript
- localStorage API
- Storage Event API
